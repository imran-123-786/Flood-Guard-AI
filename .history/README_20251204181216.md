## key features
   Feature                    Description                                      
 
   Real-Time Weather      | Fetches rainfall, humidity, alerts from live API 
   AI Flood Prediction    | Uses ML model trained on India flood dataset     
   Interactive Map        | Click any location to view risk + shelter data   
   SOS Emergency Mode     | One-tap alert + location sharing to WhatsApp     
   Safe Route Navigation  | Auto-redirects to safer elevation via maps       
   Shelter Finder         | Shows nearest flood relief centers               
   Offline Mode           | Works even if internet fails                     
   Alerts & Notifications | Auto warnings based on weather + AI              
   Community Mode         | Add volunteers & help requests                   
   Account System         | Login via Email or OTP, profile photo, settings  
   Multi-Language UI      | English + Hindi + Kannada (extendable)           
   Live Disaster News     | Automatic disaster-related RSS news feed
