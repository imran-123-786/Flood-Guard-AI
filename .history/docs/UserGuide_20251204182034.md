# User Guide

## 1️⃣ Dashboard
Shows live weather, risk level, battery, and location.

## 2️⃣ Prediction
Uses ML model to analyze risk.

## 3️⃣ Shelter Finder
Shows nearest relief centers.

## 4️⃣ SOS
Immediately sends emergency message with live location.

## 5️⃣ Offline Mode
Stores last known weather, map and shelters.
